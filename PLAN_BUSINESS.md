# Мобильное приложение SCADA для цеха десертов

Актуальность: 30.03.2026.

## Текущее состояние

- Backend работает: принимает теги от PrintSrv и устойчиво переживает временные сбои.
- Frontend работает как основной интерфейс: отображает состояние и обновляется в реальном времени.
- Android-сборка (TWA) готова и может использоваться для пилотного запуска.

Ограничение текущего этапа:

- часть бизнес-сценариев еще проходит этап шлифовки перед финальным внедрением.

## Что уже выполнено

- Построен рабочий контур данных: PrintSrv -> backend -> frontend -> Android.
- Реализовано автообновление данных без ручной перезагрузки экрана.
- Добавлены базовые механизмы оповещений и индикации состояния оборудования.
- Подготовлены Docker-сценарии для локальной и предпродовой проверки.
