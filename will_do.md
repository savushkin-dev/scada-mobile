- Оптимизировать возврат по логике: детали аппарата -> список аппаратов -> список цехов -> выход из приложения. Сейчас при возврате со страницы аппарата на страницу аппаратов, а затем на страницу цехов, а затем при попытке выйти из приложения, приложение возвращает на страницу аппаратов, а не выходит. Это связано с тем, что при открытии страницы аппарата, она добавляется в стек навигации, и при возврате со страницы аппарата, приложение возвращает на предыдущую страницу в стеке навигации, которая является страницей аппаратов. Нужно оптимизировать логику навигации так, чтобы при возврате со страницы аппарата на страницу аппаратов, страница аппаратов не добавлялась в стек навигации, а заменяла текущую страницу. Тогда при возврате со страницы аппарата на страницу аппаратов, приложение будет возвращать на страницу цехов, а при попытке выйти из приложения, приложение будет выходить.
